Node.js

HTML

HTML es el lenguaje de marcado estándar utilizado para crear páginas web. Se utiliza para definir la estructura y el contenido de las páginas web utilizando etiquetas y atributos.

CSS

CSS es un lenguaje de hojas de estilo utilizado para definir la presentación y el estilo de las páginas web. Se utiliza junto con HTML para dar estilo a la estructura y el contenido de las páginas web.

JavaScript

JavaScript es un lenguaje de programación utilizado para crear interactividad y dinamismo en las páginas web. Se utiliza principalmente en el lado del cliente para agregar funcionalidad y mejorar la experiencia del usuario.

Express

Express es un framework de Node.js utilizado para crear aplicaciones web del lado del servidor. Se utiliza para simplificar el proceso de creación de aplicaciones web y para agregar funcionalidades adicionales a Node.js.

jQuery

jQuery es una biblioteca de JavaScript utilizada para simplificar la manipulación de documentos HTML, el manejo de eventos y el desarrollo de animaciones. Se utiliza principalmente en el lado del cliente para agregar funcionalidad adicional a las páginas web.

Bootstrap

Bootstrap es un framework de CSS utilizado para diseñar y desarrollar páginas web con un diseño responsive y adaptable a diferentes tamaños de pantalla. Ofrece una gran cantidad de componentes, clases y herramientas para facilitar el desarrollo web.
